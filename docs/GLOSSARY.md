# Glossary

## Роли и права

| Термин | Описание |
|--------|----------|
| **UserRole.User** | Обычный пользователь системы |
| **UserRole.Head** | Руководитель подразделения |
| **UserRole.Admin** | Администратор системы с полными правами |
| **ChatRole.Member** | Обычный участник чата |
| **ChatRole.Admin** | Администратор чата (может управлять участниками) |
| **ChatRole.Owner** | Создатель/владелец чата (полные права) |

## Типы чатов

| Термин | Описание |
|--------|----------|
| **Chat** | Групповой чат, созданный вручную |
| **Department** | Чат отдела/подразделения (создаётся автоматически) |
| **Contact** | Личный чат 1-на-1 между двумя пользователями |
| **DepartmentHeads** | Чат руководителей подразделений |

## Сообщения

| Термин | Описание |
|--------|----------|
| **System Message** | Автоматическое сообщение о событии в чате |
| **Reply** | Ответ на конкретное сообщение (цитирование) |
| **Forward** | Пересылка сообщения из другого чата |
| **Voice Message** | Голосовое сообщение с аудиофайлом |
| **Poll** | Опрос, прикреплённый к сообщению |
| **Read Receipt** | Отметка о прочтении сообщения пользователем |
| **CreateMessageRequest** | Входной DTO для создания сообщения (отделён от выходного `MessageDto`) |

## Системные события (SystemEventType)

| Событие | Описание |
|---------|----------|
| **ChatCreated** | Чат был создан |
| **MemberAdded** | Участник добавлен в чат |
| **MemberRemoved** | Участник удалён из чата |
| **MemberLeft** | Участник покинул чат |
| **RoleChanged** | Роль участника в чате изменена |

## Транскрипция голосовых сообщений (TranscriptionStatus)

| Статус | Описание |
|--------|----------|
| **Pending** | В очереди на транскрипцию |
| **Processing** | Транскрипция выполняется |
| **Done** | Транскрипция завершена успешно |
| **Failed** | Ошибка транскрипции |

## Темы оформления (Theme)

| Значение | Описание |
|----------|----------|
| **light** | Светлая тема |
| **dark** | Тёмная тема |
| **system** | Следовать за системной темой ОС |

## Инфраструктурные термины

| Термин | Описание |
|--------|----------|
| **ChatHub** | SignalR-хаб для real-time коммуникации (`/chatHub`) |
| **HubNotifier** | Серверный сервис для отправки уведомлений через SignalR |
| **GlobalHubConnection** | Клиентское singleton-подключение к SignalR |
| **SecureStorage** | Безопасное хранилище токенов на клиенте |
| **SessionStore** | Хранилище текущей сессии (текущий пользователь, токены) |
| **Rate Limiting** | Ограничение частоты запросов (sliding window) |
| **AccessControlService** | Сервис проверки прав доступа на сервере |
| **CacheService** | Серверный in-memory кеш |
| **LocalDatabase** | Клиентская SQLite БД для оффлайн-кеширования |
